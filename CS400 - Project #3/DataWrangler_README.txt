Files Written by Me:
--------------------
distances.csv - Data of distances between intersections in a given section of UW Campus
pointOfInterests.csv - Data of points of interest at nearby each intersection in a given section of UW Campus
Intersection.java - Represents a single intersection object, with connections, and points of interests
Connection.java - Represents a single connection object, with a target intersection and distance to target
DataReader.java - Handles the data from the two files to create and return a list of intersections
DataWranglerTests.java - Tests the DataReader.java for various error prone cases
